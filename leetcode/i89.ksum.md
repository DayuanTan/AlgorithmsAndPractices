# Description

Given n distinct positive integers, integer k 
(
k
≤
n
)
(k≤n) and a number target.

Find k numbers where sum is target. Calculate how many solutions there are?


# Example
Example 1:
```
Input:

A = [1,2,3,4]
k = 2
target = 5
Output: 
2

Explanation:

1 + 4 = 2 + 3 = 5
```

Example 2:
```
Input:

A = [1,2,3,4,5]
k = 3
target = 6
Output:
1

Explanation:

There is only one method. 1 + 2 + 3 = 6
```
