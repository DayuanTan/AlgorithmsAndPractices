
912. Sort an Array
难度
中等

https://leetcode-cn.com/problems/sort-an-array/

Given an array of integers nums, sort the array in ascending order.

 

Example 1:

Input: nums = [5,2,3,1]
Output: [1,2,3,5]

Example 2:

Input: nums = [5,1,1,2,0,0]
Output: [0,0,1,1,2,5]
 
```
Constraints:

1 <= nums.length <= 5 * 104
-5 * 104 <= nums[i] <= 5 * 104
```

# solution

```java
class Solution {
    public int[] sortArray(int[] nums) {
        if (nums == null || nums.length == 0){
            return nums;
        }

        return quicksort(nums, 0, nums.length-1);
    }

    private int[] quicksort(int[] A, int start, int end){
        if (start >= end){
            return A;
        }

        int left = start;
        int right = end;
        int pivot = A[(start + end) / 2]; // is better than A[start] or A[end]; is worse than random() but expensive

        // not <, but <=
        while (left <= right){
            while (left <= right && A[left] < pivot){ // must < cannot <=
                left ++;
            }
            while (left <= right && A[right] > pivot){ // must > cannot >=
                right --;
            }

            if (left <= right){
                int temp = A[left];
                A[left] = A[right];
                A[right] = temp;
                left ++;
                right --;
            }
        }

        quicksort(A, start, right);
        quicksort(A, left, end);

        return A;
    }
}
```
